Saya merupakan mahasiswa Teknologi Maklumat di Fakulti Informatik dan Komputer (FIK), UniSZA[cite: 1]. Pembangunan laman web ini dibina khas berpandukan keperluan tugasan individu bagi kod kursus CSD34203 (Special Topics in Software Development)[cite: 1].

Fokus utama adalah mempraktikkan integrasi perisian kawalan versi Git, hos pelayan statik, dan penerokaan kerangka kod hadapan secara interaktif[cite: 1].
